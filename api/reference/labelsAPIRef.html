Labels API

Enables you to get labels for consignments and packages. You can specify label format where required.

Get Labels

Returns labels for the specified consignment.

Headers

Key Description
ocp-apim-subscription-key Your API key. Must be provided with all requests
Accept Specify what type of response you want to receive, either JSON or XML
Content-Type Specify what type of content you are sending, either JSON or XML
electio-api-version The version of the API to use, currently 1.1

Example

ocp-apim-subscription-key: [qwerrtyuiioop0987654321]
content-type: application/json
accept: application/json
electio-api-version: 1.1

Request

Parameters

GET https://api.electioapp.com/labels/{consignmentReference}

Example

Responses

Property Type Description
File Byte[] A base64-encoded byte array representing the file content
ContentType StringL The content type of the file - for example, "application/pdf".

Example

{
  "File": "SlZCRVJpMHhMalFLSmRQcjZl ... VRrNU9ERUtKU1ZGVDBZPQ==",
  "ContentType": "application/pdf"
}
