Get Labels

Returns labels for the specified consignment.

Request
GET /{consignmentReference}
Parameters
Name Type Value Notes
consignmentReference string

The consignment's unique reference.
Responses
Status Code Description Samples
200

A shipment object was returned successfully
Mime type: application/json 
{
  "type": "object",
  "properties": {
    "ContentType": {
      "type": "string"
    },
    "type": {
      "type": "string"
    },
    "message": {
      "type": "string"
    }
  },
  "x-internal-ref-name": "GetLabelsResponse"
}
