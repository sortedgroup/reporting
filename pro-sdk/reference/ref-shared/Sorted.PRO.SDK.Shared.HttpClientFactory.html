Class HttpClientFactory

GetClient(String)

Get an System.Net.Http.HttpClient for the specified baseUrl. The first call for each baseUrl will return a new HttpClient based on the url host which will then be saved for future requests. Each subsequent call to this method with the same baseUrl will return the same System.Net.Http.HttpClient. Each different baseUrl will yield a different System.Net.Http.HttpClient
